# Galeri Eksplorasi Matematik : A VR Linear Equation Simulation 

A VR-based educational application developed to help students understand linear equations through interactive balance scale simulations and immersive learning experiences. :contentReference[oaicite:1]{index=1}

## Technologies Used
- Unity3D
- C#
- Blender
- Firebase
- Meta Quest 2
- Figma

## Features
- VR interactive learning environment
- Linear equation balance scale simulation
- 3D object interaction
- Mini quiz session
- User login and registration
- Educational museum-style environment

## Project Purpose
This project was developed to improve students’ understanding of abstract linear equation concepts through visual, interactive, and immersive VR learning experiences. :contentReference[oaicite:2]{index=2}

## Development Methodology
- ADDIE Model
- User-centered VR learning design
